# 电商用户行为数据分析项目
基于真实电商数据，使用 Python 进行用户行为分析、转化漏斗、用户分层、品类分析，输出可落地业务建议。

## 项目介绍
本项目通过对电商用户「浏览、收藏、加购、购买」行为数据进行分析，完成
- 24 小时行为分布分析
- 商品品类转化率分析
- 用户 RF 分层（最近购买 + 购买频次）
- 全链路漏斗转化分析

最终定位业务瓶颈、输出优化策略，完整复现电商数据分析实战流程。

## 技术栈
- Python
- Pandas：数据清洗、分组统计、时间处理
- Matplotlib：可视化绘图（时段图、漏斗图、用户分层图）
- Excel：自动输出报表结果

## 分析内容
### 1. 24 小时用户行为高峰分析
统计各小时用户行为，识别高峰时段，指导分时运营。

### 2. 商品品类转化率分析
按类别统计浏览量、购买量、转化率，定位爆款、潜力品类、低效品类。

### 3. 用户 RF 分层模型
基于最近购买时间 & 购买频次，将用户分为：
- 高价值用户
- 潜力用户
- 沉睡用户
- 流失用户

### 4. 全链路漏斗转化分析
构建「浏览 → 收藏 → 加购 → 购买」转化漏斗，定位流失环节。

## 项目成果
- 识别平台流量高峰时段，支持分时运营
- 挖掘高转化潜力品类，优化品类结构
- 完成用户 4 层级分层，可直接用于精细化运营
- 漏斗分析明确核心流失点，给出可落地优化方案
- 自动生成图表 + Excel 报表
